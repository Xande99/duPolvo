Labeled input / textarea with hint, required, error and dark-surface support.

```jsx
<Input label="Nome" placeholder="Seu nome" required />
<Input label="Mensagem" multiline placeholder="Conte sobre seu projeto" />
<Input label="E-mail" error hint="E-mail inválido" onDark />
```

Set `multiline` for a textarea. Use `onDark` inside navy CTA sections. `error` + `hint` shows a red validation message.
